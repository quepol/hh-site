# hh-site

Source for [hillaryhartley.com](https://hillaryhartley.com) — a one-page personal site for Hillary Hartley, working at the intersection of technology, democracy, and the people they both serve.

## Stack

Plain HTML and CSS. No frameworks, no build step, no dependencies. Hosted on GitHub Pages with a custom domain via `CNAME`.

## Local preview

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server
```

## Deployment

Push to `main` and GitHub Pages handles the rest.
